## 核心知识点

### 3D 打印

| 耗材 | 特点 | 适用场景 |
|------|------|----------|
| PLA | 易打印、无毒、脆 | 原型、外观件 |
| PETG | 韧性好、耐温略高 | 功能件 |
| ABS | 强度高、需封箱 | 承力件 |
| TPU | 柔性 | 缓冲、轮胎 |
| 光敏树脂 | 精度高、有气味 | 精细小件 |

**切片关键参数**：层厚、壁厚、填充密度、支撑、 brim/raft、打印温度、速度。

### CNC 雕刻机

1. **软件安装**：USB 驱动 → CNC V4 控制软件 → 精雕/NC 转换器；
2. **刀路生成**：在精雕软件中绘制图形 → 生成刀路 → 导出 `.eng` → 转换为 `.nc`；
3. **软件端操作**：连接机器 → 回机械原点 → 对刀 → 加载 NC → 启动加工；
4. **安全要点**：佩戴护目镜、固定工件、先空跑再加工、禁止戴手套操作旋转刀具。

### 激光雕刻机

- 使用前检查光路、对焦与排烟；
- 根据材料设置功率与速度；
- 严禁雕刻不明材料（如 PVC 会释放有毒气体）。

## 实践任务

### 任务 1：3D 打印切片

使用 [Day 3](/training/day03-mechanical-modeling) 导出的 **STL** 文件：

1. 在 BambuLab 或 Creality Print 中导入模型；
2. 设置合理层厚（0.2mm）、填充密度（20%）、支撑（按需）；
3. 生成切片并导出 G-code；
4. 截图保存切片预览与预计时间。

### 任务 2：CNC 雕刻流程演练

在指导人员陪同下完成：

1. 观看 CNC 软件安装与刀路生成演示；
2. 记录对刀步骤（X/Y 定位、Z 对刀）；
3. 填写设备使用安全检查表。

### 任务 3：激光雕刻安全学习

阅读 [Falcon2 指南](/mechanical/falcon2-laser-engraver) 的安全章节，列出 3 条必须遵守的安全规则。
