## 核心知识点

### 三种常见底盘

| 底盘类型 | 优点 | 缺点 | 典型应用 |
|----------|------|------|----------|
| 差速底盘 | 结构简单、成本低 | 不能横移 | 竞速、搬运 |
| 全向轮底盘 | 可全向移动、控制直观 | 越障弱、轮子多 | 物流、舞蹈 |
| 麦轮底盘 | 可全向移动、布局紧凑 | 精度要求高、磨损 | 竞赛机器人 |

### 麦轮 X 型布局速度映射

对于常见 X 型布局，四个轮子的转速 \(v_1, v_2, v_3, v_4\) 与机器人速度 \(v_x, v_y, \omega\) 的关系：

```
v1 =  vx - vy - ω
v2 = -vx - vy - ω
v3 = -vx + vy - ω
v4 =  vx + vy - ω
```

> 实际系数与轮距、轮半径有关，以上为简化示意。详细推导见 [麦轮底盘](/mechanical/chassis-mecanum)。

### 机构设计 checklist

- [ ] 明确机构需要完成的动作（移动、抓取、抬升、翻转等）；
- [ ] 确定驱动方式（电机、气缸、电磁铁等）；
- [ ] 设计运动副与约束（铰接、滑轨、凸轮等）；
- [ ] 进行干涉检查与关键受力分析；
- [ ] 预留传感器与线束安装空间。

## 实践任务

### 任务 1：底盘选型分析

假设你的机器人需要完成以下赛季任务，请选择一种底盘并说明理由：

1. 在平坦场地快速取放物品；
2. 在狭窄区域精确定位并旋转；
3. 跨越 2cm 高的门槛并直线冲刺。

### 任务 2：运动学计算

给定麦轮底盘参数：

- 轮半径 \(r = 0.05m\)
- 机器人中心到轮子距离 \(L = 0.15m\)
- 期望速度：\(v_x = 0.3m/s, v_y = 0.2m/s, \omega = 0.5rad/s\)

请计算四个轮子的目标转速（rpm）。

### 任务 3：机构简图

为你的赛季机器人绘制一张机构简图，包含：

- 底盘布局（俯视图）；
- 至少 1 个执行机构（如抬升、夹取）的运动示意；
- 关键尺寸标注。
