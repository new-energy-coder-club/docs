## 核心知识点

### 1. 标准件分类

| 类别 | 典型零件 |
|------|----------|
| 紧固件 | 螺栓、螺母、垫片、销、键、铆钉 |
| 传动件 | 齿轮、同步带轮、链轮、丝杠、联轴器 |
| 轴系与导向 | 光轴、轴承、直线导轨、轴套 |
| 结构件 | 铝管、型材、角码 |
| 动力件 | 各类电机与减速器 |
| 气动与连接 | 气缸、电磁阀、气管、吸盘、扣件 |

### 2. 常用材料速查

| 材料 | 主要特点 | 典型应用 |
|------|----------|----------|
| 6061 铝合金 | 中等强度、耐腐蚀、易加工 | 结构件、框架 |
| 7075 铝合金 | 强度高、机械性能好 | 高应力结构件 |
| 45# 钢 | 综合性能好 | 通用结构件 |
| 304 不锈钢 | 耐腐蚀 | 潮湿或外观件 |
| POM | 自润滑、耐磨 | 齿轮、防撞件 |
| PLA / PETG | 易打印、成本低 | 原型、非承力件 |
| 碳纤维 | 强度高、重量轻、贵 | 轻量化关键结构 |

### 3. 工艺选择原则

- **车削 / 铣削**：适合金属件，精度高、成本中等；
- **CNC**：复杂金属零件首选；
- **3D 打印（FDM）**：快速原型，适合验证结构；
- **光固化（SLA）**：高精度小零件；
- **激光切割**：薄板件，便宜快速；
- **钣金**：薄板结构，适合批量。

## 实践任务

### 任务 1：整理标准件清单

为你所在的小组整理一份**常用标准件清单**，至少包含：

- 3 种紧固件（含规格与用途）；
- 2 种传动件；
- 2 种轴承或导向件。

### 任务 2：材料与工艺选型

从以下场景中选择 2 个，说明你会选用的材料与加工工艺，并简述理由：

1. 机器人框架主梁（需承力、轻量化）；
2. 快速验证用的支架原型；
3. 耐磨齿轮；
4. 外观罩壳。

列出你今天可能使用的 3 种工具，并写出对应的安全注意事项。
