## 实践任务

### 任务 1：加入协作工具

- 加入 NEC 飞书组织与对应方向群；
- 关注 B 站培训录制账号；
- 在 GitHub 上 fork [`new-energy-coder-club/docs`](https://github.com/new-energy-coder-club/docs)。

### 任务 2：配置本地开发环境

```bash
# 克隆文档仓库
git clone https://github.com/new-energy-coder-club/docs.git
cd docs

# 安装 Mintlify CLI（如未安装）
npm install -g mintlify

# 本地预览
mint dev
```

### 任务 3：提交第一个 PR

1. 在 [`start-here/first-good-issue`](/start-here/first-good-issue) 中找一个适合你当前状态的任务；
2. 创建分支：`git checkout -b feat/day01-yourname`；
3. 完成修改后运行 `python tools/ci/check_docs.py`；
4. 提交 PR，标题格式：`docs: 简短描述（#issue 编号）`。
