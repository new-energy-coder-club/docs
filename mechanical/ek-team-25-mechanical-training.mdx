## Day 1：机器人标准件常用件介绍

**类型**：通识课

**建议时长**：2 小时

**学习目标**：

- 认识机器人中常用的标准件与常用件；
- 理解不同零件的功能与典型应用场景；
- 能够在设计初期进行合理的零件选型。

### 1.1 紧固件

螺栓、螺母、垫片、弹簧垫片、防松垫片、铜柱（铝、钢、尼龙）、键、销、铆钉（热、冷、抽芯、击芯）、涨套。

### 1.2 传动件

齿轮（柱、锥、直、斜、弧形）、皮带（轮）、同步带（轮）、链（轮）、蜗轮蜗杆、丝杠（滚珠）。

### 1.3 轴系与导向

光轴、轴座（T 型、L 型）、光轴固定环、轴套（铜、钢）、石墨轴套（法兰、垫片）、轴承（深沟球、法兰、角接触、推力球、直线、双列、调心、滚子、圆锥滚子、滚针、交叉滚子、回转支承等）、直线导轨（滑块）、拖链。

### 1.4 动力件

电机（旋翼、关节、云台、直线、步进、反馈）、联轴器（梅花、刚性、法兰、十字、螺旋线、平行线、磁力、万向节）。

### 1.5 结构件

铝管、型材（T 型、船型、铸造角码、挤压角码）。

### 1.6 气动与连接

气动元件（瓶、气缸、气泵、气管、接头、电磁阀、吸盘）、扣（搭扣、球扣、磁扣）、电磁铁（方、圆、圆空心）、脚轮、全向轮、麦轮、弹簧（拉、压、卷、扭）、气弹簧（液压杆）、滑环（电、气、电气、空心、微型）、直角连接件（角铁、机加工直角连接件、六面螺母）等。

### 今日任务

1. 整理一份自己队伍常用的标准件清单；
2. 挑选 3 种不熟悉的零件，查阅其安装方式与选型参数。

## Day 2：常见材料、加工工艺与机器人装配常用工具

**类型**：通识课

**建议时长**：2 小时

**学习目标**：

- 了解机器人常用材料的特性与适用场景；
- 熟悉常见制造工艺及其精度、成本特点；
- 掌握实验室常用工具的安全使用方法。

### 2.1 常见材料

#### 金属材料

| 材料 | 主要特点 | 典型应用 |
| --- | --- | --- |
| 黄铜 | 耐腐蚀耐热、强度高、导电性好、较软、贵 | 丝杠螺母 |
| 6061 铝合金 | 中等强度、抗腐蚀、可焊接、氧化效果较好 | 结构件、框架 |
| 7075 铝合金 | 强度高、机械性能好、阳极反应良好 | 高应力结构件 |
| 5052 铝合金 | 抗疲劳、塑性与耐腐蚀性高、焊接性好 | 钣金 |
| 45# 钢 | 综合性能好，强度硬度韧性均衡 | 通用结构件 |
| 碳钢 | 强度高、成本低 | 螺纹连接件 |
| Cr 钢 | 耐热、高硬度、高抗氧化 | 轴承、直线导轨 |
| 304 不锈钢 | 综合性能良好、耐腐蚀 | 潮湿或外观件 |
| 弹簧钢 | 韧性好、强度高、加工性差 | 弹簧、弹性元件 |

#### 非金属材料

| 材料 | 主要特点 | 典型应用 |
| --- | --- | --- |
| 亚克力（PMMA） | 便宜、透明、加工性好 | 罩壳、观察窗 |
| 玻纤 | 便宜、强度尚可、较重 | 板件 |
| 碳纤维 | 强度高、刚度高、很轻、贵 | 轻量化结构 |
| POM | 自润滑、耐磨 | 齿轮、防撞 |
| 高密度 PE | 耐磨、自润滑、无毒 | 耐磨件 |
| PP | 类似 PE，略重 | 一般结构件 |
| PA（尼龙） | 弹性与刚性可调、常用 | 齿轮、结构件 |
| PLA | 易打印、无毒、保质期短 | 3D 打印原型 |
| PETG | 比 PLA 韧性好、打印难度略高 | 3D 打印件 |
| ABS | 强度高、韧性好、需封箱打印 | 功能件 |
| TPU | 柔性、弹性好 | 柔性结构 |
| 光敏树脂 | 精度高、有微毒气体 | 光固化打印 |

> **纤维类小贴士**：纤维类材料一般加工工艺为板材切割、吹塑、包覆，并且为各向异性材料，使用时要注意减少切断纤维的数量。

### 2.2 常见工艺

| 工艺 | 特点 | 适用场景 |
| --- | --- | --- |
| 车削 | 工件旋转、车刀进给 | 轴类、盘类零件 |
| 铣削 | 旋转多刃刀具切削 | 平面、曲面 |
| 钻削 | 孔加工基本方法 | 各类孔 |
| 磨削 | 高精度 | 精密表面 |
| 拉削 | 高效率、尺寸精度高 | 内键槽等 |
| 锉削 | 手工修整 | 复杂表面、装配修整 |
| 制齿 | 铣齿、滚齿、插齿等 | 齿轮 |
| CNC | 计算机数字化控制 | 复杂零件 |
| 挤压/拉拔 | 连续截面 | 管材、型材 |
| 钣金 | 冷加工、精度有限、便宜 | 薄板件 |
| 激光切割 | 切缝窄、便宜 | 板件 |
| 焊接 | 接合金属或塑料 | 框架、支架 |
| 冲压 | 大批量、成本低 | 小零件 |
| 铸造 | 工艺粗糙、价格低 | 复杂外形 |
| 锻造 | 改善金属组织 | 高强度件 |

## Day 3：SolidWorks 与 AutoCAD 软件简介与设置

**类型**：软件课

**建议时长**：2 小时

**软件要求**：SolidWorks 2023，AutoCAD 2018 及以上。

**学习目标**：

- 完成 SolidWorks 基础环境配置；
- 掌握草图、特征、装配体等基本命令；
- 了解 AutoCAD 在队内的使用场景。

### 3.1 SolidWorks 基础

- **设置**：正视于（Spacebar）、电脑配置不行的相关设置。
- **快捷键设置**：可以教学，具体视大家接受度而定。
- **草图教学**：直线、矩形、圆形、正多边形、中心线、线性阵列、圆周阵列、镜像、完全定义。
- **特征命令教学**：拉伸、旋转、扫描、放样（及对应切除）、圆角、倒角、线性阵列、圆周阵列、镜像、参考几何体、螺旋线。
- **直接编辑**：分割、组合、移动/复制实体、删除/保留实体、移动面、删除面。
- **装配体**：插入新零件、线性阵列、圆周阵列、镜像。
- **配合**：重合、平行、垂直、同心、距离、角度。
- **其他**：随配合复制、替换零部件、干涉检查、质量属性、导出 DWG、导出 STL。

### 3.2 AutoCAD

队内使用相对较少，以自学为主。

### 今日任务

## Day 5：运动分析、力分析与机构

**类型**：设计基础

**建议时长**：4 小时

**学习目标**：

- 建立静力学与动力学分析基础；
- 理解常见机构与传动方式；
- 能够用能量法进行简单设计计算。

### 5.1 静力学分析

- 力、静力学公理（平行四边形法则、二力平衡、作用力与反作用力、刚体的力转移、三力平衡汇交）
- 约束（固定铰链、可动铰链、特定形状约束、二力杆）
- 力矩相关（牛顿第二定律、旋转物体的动能、转动惯量）
- 力偶（一对反向平行不共线的力，力偶矩）
- 自锁

### 5.2 动力学分析

- 能量与动量的关系
- 转动物体的动能计算
- 机械能守恒
- 以能量守恒（或动量守恒）入题进行设计计算

### 5.3 机构分析

- 自由度
- 常见机构（连杆、凸轮）
- 常见传动方式（链、带、摩擦、螺旋、齿轮、涡轮、减速箱、绳驱）
- 常用动/静连接方式（键、销、螺栓、联轴器、涨紧、传动机构润滑）

### 今日任务

1. 计算一个简单机构的自由度；
2. 用能量法估算某一传动环节所需的电机功率。

## Day 7：经济性优化

**类型**：设计基础

**建议时长**：1 小时

**学习目标**：

- 了解零件加工费用的组成部分；
- 掌握典型零件（板件、铝管、打印件、机加工件）的成本估算方法；
- 在不影响效果的前提下优化设计成本。

### 7.1 加工费用组成

材料、机床、编程、装夹、加工、后处理、工时费、数量折扣。

### 7.2 典型零件成本估算

| 零件类型 | 计费方式 | 备注 |
| --- | --- | --- |
| 板件 | 主要算面积，辅助算特征数量 | — |
| 铝管 | 截面外形尺寸和长度 | 特征多会加钱 |
| 打印件 | 算克重 | FDM 约 0.2 元/克，铝合金约 2 元/克 |
| 机加工件 | 材料 + 机床 + 编程 + 装夹 + 加工 + 后处理 | 各厂家算法不同 |

### 7.3 以嘉立创 CNC 为例

- 机床费用算进加工费；
- 编程费用为工程费；
- 影响加工费的因素：工序（装夹、换机床次数）、材料加工难度、零件本身设计特征。

### 7.4 核心要点

经济性优化只是为省钱做出的让步，精髓是在满足设计余量的情况下追求极致的性价比。核心是不影响效果的情况下追求便宜，不要舍本逐末。

**实例**：对比 1.2 的舵轮驱动轮系和 1.3 的舵轮驱动轮系。

### 今日任务

1. 估算当前设计中的 3 个零件的加工成本；
2. 提出至少 1 处不影响性能的成本优化方案。

## Day 10：结营考核 / 综合设计

**类型**：考核

**建议时长**：约 4 周（11 月 9 日—12 月 7 日）

**考核目标**：

- 完成一个完整的设计方案；
- 能够给出预计报价；
- 锻炼团队协作与综合设计能力。

### 设计题目

一个物流运输场景，需要将一件物品从 A 位置移动到 B 位置，并改变物体的姿态。三选一：

- **题目一**：位置不可移动，仅允许选择一个位置为起点，另一位置为终点。
- **题目二**：不要求位置变化（是否移动都行），但必须满足姿态变换。
- **题目三**：位置可以移动，满足台面高低差和姿态变换即可。

### 要求
