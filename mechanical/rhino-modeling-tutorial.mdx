## 二、建模操作教程

### （一）基础准备

1. **设置尺度**：将整体的尺度设置为合适单位，如米。
2. **确定视图**：以三维视图为主要操作视图，其他视图为参考视图。
3. **插入参考点**：点击“点”，输入坐标 `(x,y,z)` 自动插入参考点，例如插入 `A(6,0,0)` 和 `B(0,0,0)`。

### （二）二维草图绘制

#### 1. 绘制线条

| 命令 | 说明 |
| --- | --- |
| **Line（直线）** | 通过指定起点和终点绘制直线。 |
| **Polyline（多重直线）** | 绘制连续的直线段。 |
| **Arc（圆弧）** | 通过起点、终点、控制点或角度/半径绘制圆弧。 |
| **Rectangle（矩形）** | 指定角点绘制矩形，也支持圆角矩形（Rounded）。 |
| **Ellipse（椭圆）** | 指定中心点、轴和旋转角度绘制椭圆。 |
| **Polygon（多边形）** | 指定边数和半径或中心点创建规则多边形。 |

#### 2. 辅助操作

在绘制过程中，配合使用**对象捕捉**功能可让操作更精确。常用的捕捉模式包括：

- 端点捕捉
- 中点捕捉
- 圆心捕捉
- 交点捕捉

可通过状态栏的捕捉按钮或快捷键操作，例如：

- `F3`：切换端点捕捉
- `F6`：切换中点捕捉

### （三）三维建模

#### 1. 拉伸形成基本体

选中绘制好的封闭曲线，使用 `ExtrudeCrv` 命令（或菜单：`Solid > Extrude Curve > Straight`），输入高度值，即可得到基本立体图形，如正方体。

#### 2. 倒角处理

使用 `FilletEdge` 命令，选择想要倒角的棱边，输入半径值，可让棱角变圆滑。

#### 3. 布尔运算

| 命令 | 用途 | 操作要点 |
| --- | --- | --- |
| **BooleanDifference** | 创建凹槽 | 先选择主模型，再选择要减去的模型，执行后生成凹槽。 |
| **BooleanUnion** | 合并形体 | 将多个相交的实体合并为一个整体。 |

#### 4. 曲面创建

| 命令 | 说明 |
| --- | --- |
| **单轨扫掠** | 选择扫掠路径和扫掠形状，生成曲面。 |
| **旋转成型** | 选择待旋转的曲线和旋转轴，设置旋转度数后按 `Enter` 生成曲面。 |

#### 5. 加盖操作

若模型顶部或底部未封闭，可使用 `Cap` 命令，选中开口的实体或面体，Rhino 会自动为开放区域加盖。
