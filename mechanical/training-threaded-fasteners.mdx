## 文档简介

螺纹系零件是机器人装配中最常用也最容易出错的零件。本文档系统梳理螺栓、螺母、螺钉与铜柱的分类、选型方法与安装细节，并给出滑牙、锈死等"急救"方案。

## 螺栓与螺母

- **螺丝**是所有带外螺纹紧固件的俗称；标准文件只分 Bolt（螺栓）、Screw（螺钉）和 Stud（螺柱）；
- **螺栓**：带外螺纹的紧固件，通常与螺母配合使用，连接两个或多个带通孔的零件；按头部形状分六角头、圆头、方头、沉头等。

### 常用螺母类型

| 类型 | 特点与用途 |
|---|---|
| 六角螺母 | 最常见，结构强度高，广泛用于机械、建筑、汽车 |
| 法兰螺母 | 一端带法兰盘，省去垫圈，防松、分散压力，适用振动环境 |
| 盖形螺母 | 顶端封闭，保护螺纹、美观防锈，用于装饰或户外设备 |
| 自锁螺母 | 带尼龙圈或金属变形结构，防松脱，用于高铁、发动机等振动场合 |
| T 型螺母 | 嵌入铝型材或木材中，用于家具、CNC 工装夹具 |
| 桶形螺母 | 圆柱形隐藏式安装，用于家具、航空结构连接 |
| 耦合螺母 | 长型全螺纹，连接两根螺杆或延长螺纹杆 |
| 笼式螺母 | 卡扣结构快速装拆，常用于服务器机柜、电控箱 |
| 焊接螺母 | 金属板焊接固定，常见于汽车制造 |

### 防松措施

1. 垫片、弹簧垫片、防松垫片；2. 双螺母；3. 螺纹胶（如 Loctite 243）。

### 安装细节

1. 螺栓与螺母配套使用（M4 螺栓 + M4 螺母 + M4 垫片，特殊情况垫片可适当大一号）；
2. **"先手拧，后对角"**：手动拧到底，再对角线分两阶段拧紧（有电动工具可直接加对应型号套筒）；
3. **"自锁螺母一次性"**：尼龙锁紧螺母拆过 3 次必须换新；
4. **急救包**——螺母滑牙：小一号螺纹攻丝（M8 滑牙改 M10），或塞钢丝螺套（Helicoil）恢复原规格；锈死螺母：喷 WD-40 → 轻敲震动 → 加长扳手瞬间拧松，或气割螺母（螺栓报废，仅应急）。

## 螺钉

- 俗称螺丝、螺丝钉，通常不需要螺母，直接与被连接件配合（一般先钻孔攻螺纹）；工具多用螺丝刀，头部一字槽、十字槽、内外六角常见。

### 选型三步法

1. **看孔型**：已攻丝盲孔 → 螺钉；无螺纹通孔 → 螺栓 + 螺母；薄板无螺纹 → 自攻钉；
2. **看空间**：头部需隐藏 → 沉头或半沉头；需高扭矩 → 内六角圆柱头；
3. **看材质/环境**：普通室内 → 镀锌碳钢；户外/潮湿 → 不锈钢 A2-70；高强度振动 → 10.9 级合金钢 + 螺纹胶。

### 安装注意（防松、防断、防花）

1. 螺纹孔清洁：去毛刺、去铝屑，避免假扭矩；
2. 预紧顺序：先手动对牙，再对角分两次拧紧；
3. 内六角别用错：S2 合金钢批头 + 垂直压紧，防止"圆角"；
4. 紧定螺钉：尖端必须顶在轴平面或顶丝孔，**禁止顶弧面**。

## 铜柱（铝、钢、尼龙）

### 选型三步法（30 秒完成）

1. **定螺纹与孔径**：主板/子板常用 M3（孔径 3.2 mm）；精密模块用 M2（孔径 2.2 mm）；
2. **定高度**：单面板高 = 板厚 + 元件高度 + 1 mm 余量；叠板直接量两板净距；
3. **选结构**：双通内螺纹（最常用，拆装方便）；单通外/内螺纹（一端机箱一端主板）；SMT 贴片铜柱（回流焊，适合自动化）。

### 材质对比

- **黄铜镀镍**：导电/导热好，外观亮，最常用；
- **不锈钢 A2**：耐腐蚀，但单价贵约 30%；
- **尼龙 + 铜嵌件**：绝缘、轻量，但只能手拧。
