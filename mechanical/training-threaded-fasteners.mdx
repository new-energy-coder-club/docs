螺丝：

螺丝”是所有带外螺纹紧固件的俗称，包括螺栓、螺钉、自攻钉等；标准文件里其实只

分 Bolt（螺栓）、Screw（螺钉） 和 Stud（螺柱）

螺栓：

定义：一种带外螺纹的紧固件，通常与螺母配合使用，用于连接两个或多个带通孔的

零件。按头部形状分类有：六角头、圆头、方头、沉头等。

![机械组培训：螺纹系零件 插图](https://cdn.newenergycoder.club/files/mech-training/images/mech-training-threaded-fasteners/p01-img02.jpeg)

螺母：

|    类型     |               特点与用途说明                                |

| **六角螺母** | 最常见，结构强度高，广泛用于机械、建筑、汽车等领域。        |

| **法兰螺母** | 一端带法兰盘，省去垫圈，防松、分散压力，适用于振动环境。    |

| **盖形螺母** | 顶端封闭，保护螺纹、美观防锈，常用于装饰或户外设备。        |

| **自锁螺母** | 带尼龙圈或金属变形结构，防松脱，用于高铁、发动机等振动场合。|

| **T 型螺母** | 嵌入铝型材或木材中，用于家具、CNC 工装夹具等。              |

| **桶形螺母** | 圆柱形，隐藏式安装，用于家具、航空结构连接。                |

| **耦合螺母** | 长型全螺纹，用于连接两根螺杆或延长螺纹杆。                  |

| **笼式螺母** | 卡扣结构，便于快速安装/拆卸，常用于服务器机柜、电控箱。     |

| **焊接螺母** | 用于金属板焊接固定，常见于汽车制造。                        |

防松措施：1.垫⽚、弹簧垫⽚、防松垫⽚  2.双螺母   3.螺纹胶（如Loctite 243）

安装细节：

1、螺栓需要与螺母配套使用（M4 螺栓M4 螺母M4 垫⽚，特殊情况下垫⽚可适当大

一号）

2、“先手拧，后对角”：手动拧到底，再对角线分两阶段拧紧

##有电动工具可直接加对应型号套筒，锁紧，安装方便省力##

3、“自锁螺母一次性”：尼龙锁紧螺母拆过3 次必须换新（大部分防滑螺母都要）

4、“急救包“

•螺母滑牙：

o小一号螺纹攻丝 → M8 滑牙改M10。

o或塞钢丝螺套（Helicoil）→ 恢复M8 原规格。

•锈死螺母：

o喷WD-40 → 轻敲震动 → 用加长扳手瞬间拧松。

o或气割螺母（螺栓报废，仅应急）。

螺钉：

俗称螺丝、螺丝钉，通常不需要螺母，直接与两个物体相配合（一般连接件先要钻

孔，攻螺纹），工具一般用螺丝刀（改锥），头部多为一字槽，十字槽，内外六角也很

多，一般较小。有些是尖头的像钉子一样，有些是平头的如同螺柱一样。

![机械组培训：螺纹系零件 插图](https://cdn.newenergycoder.club/files/mech-training/images/mech-training-threaded-fasteners/p03-img05.jpeg)

![机械组培训：螺纹系零件 插图](https://cdn.newenergycoder.club/files/mech-training/images/mech-training-threaded-fasteners/p04-img06.png)

1、选型三步法

1. 看孔型

• 已攻丝盲孔 → 用“螺钉”

• 无螺纹通孔 → 用“螺栓+螺母”

• 薄板无螺纹 → 用“自攻钉”

2. 看空间

• 头部需隐藏 → 选沉头或半沉头

• 需高扭矩 → 选内六角圆柱头

3. 看材质/环境

• 普通室内 → 镀锌碳钢

• 户外/潮湿 → 不锈钢 A2-70

• 高强度振动 → 10.9 级合金钢 + 螺纹胶

2、安装注意（防松、防断、防花）

1、螺纹孔清洁：去毛刺、去铝屑，避免假扭矩。

2、预紧顺序：先手动对牙，再对角分两次拧紧。

3、内六角别用错：S2 合金钢批头 + 垂直压紧，防止“圆角”。

4、紧定螺钉：尖端必须顶在轴平面或顶丝孔，禁止顶弧面。

铜柱（铝、钢、尼龙）：

1、选型三步法（30 秒完成）

1. 确定螺纹与孔径

– 主板/子板常用 M3（孔径 3.2 mm）。

– 精密模块用 M2（孔径 2.2 mm）。

2. 确定高度

– 单面板高：板厚 + 元件高度 + 1 mm 余量。

– 叠板：直接量两板之间净距。

3. 选结构

– 双通内螺纹：上下都是螺钉，拆装方便（最常用）。

– 单通外螺纹/内螺纹：一端拧机箱，一端拧主板。

– SMT 贴片铜柱：回流焊直接焊在 PCB 上，适合自动化。

![机械组培训：螺纹系零件 插图](https://cdn.newenergycoder.club/files/mech-training/images/mech-training-threaded-fasteners/p05-img07.png)

2、材质对比

•黄铜镀镍：导电/导热好，外观亮，最常用。

•不锈钢 A2：耐腐蚀，但单价贵 30 %。

•尼龙 + 铜嵌件：绝缘、轻量，但只能手拧
