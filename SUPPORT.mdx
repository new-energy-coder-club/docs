# 求助与支持（Support）

## 支持渠道

| 渠道 | 适用场景 | 响应时间 |
| :--- | :--- | :--- |
| **Gitee Issues** | 代码、文档、任务类问题 | 1–3 个工作日 |
| **社区 QQ/飞书群** | 快速提问、实时讨论 | 视在线成员而定 |
| **周会/例会** | 阻塞性、跨 SIG 问题 | 当周同步 |
| **邮件/Maintainer 私信** | 隐私或安全相关事项 | 24 小时内确认 |

## 提问前请自查

在提问前，建议先完成以下步骤，通常能解决 80% 的常见问题：

1. 通读 [快速开始](/start-here/index) 与相关 SIG 文档。
2. 在 Gitee Issues 和飞书 Wiki 中搜索关键词，确认是否已有类似问题。
3. 检查环境版本（Python、Git、Node、平台工具链）是否满足要求。
4. 尝试最小复现：把问题简化到最短命令或最小代码片段。

## 有效提问模板

提交 Issue 或群聊提问时，请尽量包含以下信息：

- **目标**：你想实现什么？
- **环境**：操作系统、软件版本、分支/提交号。
- **已执行命令**：完整可复制粘贴的命令行。
- **实际输出**：完整报错日志或截图。
- **已尝试步骤**：你做过哪些排查，结果如何？

## 我们不保证的

- 即时 1 对 1 答疑：社区成员多为志愿者，会尽力帮助但无法保证秒回。
- 代写代码/作业：我们鼓励引导式教学，而非直接代替完成。
- 商业技术支持：如需深度企业级支持，可关注 NEC+ 会员服务。
