## 当前阶段概览

| 阶段 | 时间 | 核心目标 | 关键交付物 |
| :--- | :--- | :--- | :--- |
| **0 阶段：基础设施** | 1–3 天 | 建立社区入口与协作规范 | 根 README 三入口、docs 站点骨架、Issue/PR 模板 |
| **1 阶段：新人 onboarding** | 2 周 | 让新成员 60 分钟内完成第一次贡献 | 新人 60 分钟教程、First Good Issue、最小 Demo、Support 页 |
| **2 阶段：项目模板化** | 1–2 月 | 沉淀可复用的竞赛与项目文档 | 重点项目页、竞赛模板、每周短复盘机制 |
| **3 阶段：生态与资产化** | 持续 | 将经验转化为可售卖/可传播的资产 | NEC+ 内测、模板库、作品集、成功案例 |
