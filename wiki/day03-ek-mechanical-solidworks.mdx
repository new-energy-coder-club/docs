# Day03 · EK 战队机械：SolidWorks 与 AutoCAD 基础及设置

> 本教程基于 [EK 战队机械组十日集训](/mechanical/ek-team-25-mechanical-training) Day 3 内容细化，面向机器人机械设计新人，重点掌握 SolidWorks 基础建模与制图流程。

---

## 1. 今日目标

- 完成 SolidWorks 2023 基础环境配置；
- 掌握草图、特征、装配体等核心命令；
- 了解 AutoCAD 在队内的使用场景；
- 完成一个带孔支架的建模，并导出 STL 与 DWG。

**建议时长**：2 小时  
**软件版本**：SolidWorks 2023，AutoCAD 2018 及以上。

## 2. 软件安装与环境配置

### 2.1 SolidWorks 2023 安装要点

1. 确认电脑配置满足官方最低要求（推荐 16 GB 内存、独立显卡）。
2. 安装时选择**单机安装**，序列号由队内统一提供。
3. 安装完成后首次启动，选择**新建零件**进入工作界面。

### 2.2 必做基础设置

| 设置项 | 推荐配置 | 说明 |
| --- | --- | --- |
| 正视于快捷键 | `Spacebar` | 快速摆正视角 |
| 单位制 | `MMGS`（毫米、克、秒） | 与国内加工单位一致 |
| 默认模板 | ISO 模板 | 工程图与国标接近 |
| 自动保存 | 10 分钟 | 防止崩溃丢失 |
| 备份文件 | 启用 | 保留 `.sldprt` 历史版本 |

> 如果电脑配置较低，可在**系统选项 > 性能**中关闭 RealView 和阴影，提升操作流畅度。

## 3. SolidWorks 核心命令

### 3.1 草图命令

进入草图环境的两种方式：

- 选中基准面 → 点击**草图绘制**；
- 直接点击某个特征，SolidWorks 会自动进入草图。

常用草图命令：

| 命令 | 图标/快捷键 | 用途 |
| --- | --- | --- |
| 直线 | `L` | 绘制直线段 |
| 矩形 | `R` | 边角矩形、中心矩形 |
| 圆 | `C` | 圆心圆、周边圆 |
| 多边形 | — | 正多边形 |
| 中心线 | — | 构造线、对称轴 |
| 线性阵列 | — | 沿方向复制草图实体 |
| 圆周阵列 | — | 绕中心复制草图实体 |
| 镜像 | — | 沿中心线镜像 |
| 智能尺寸 | `D` | 标注尺寸并驱动几何 |

**草图完全定义**：线条由黑色变为蓝色表示欠定义，黑色表示完全定义。设计时尽量保持完全定义，避免模型意外变化。

### 3.2 特征命令

| 命令 | 用途 | 常用场景 |
| --- | --- | --- |
| 拉伸凸台/基体 | 将草图沿法向拉伸出实体 | 板件、支架 |
| 拉伸切除 | 用草图切去材料 | 开孔、挖槽 |
| 旋转凸台 | 草图绕轴旋转生成回转体 | 轴套、轮子 |
| 旋转切除 | 旋转切除材料 | 内腔、倒角 |
| 扫描 | 轮廓沿路径移动生成实体 | 异形管道 |
| 放样 | 多个截面之间过渡生成实体 | 复杂曲面 |
| 圆角 | 对边倒圆角 | 去毛刺、美观 |
| 倒角 | 对边倒角 | 装配定位、去毛刺 |
| 线性/圆周阵列 | 复制特征 | 重复孔、肋板 |
| 镜像特征 | 对称复制特征 | 对称结构 |
| 参考几何体 | 创建基准面、基准轴、坐标点 | 辅助建模 |
| 螺旋线 | 创建弹簧、螺纹扫描路径 | 弹簧、螺杆 |

### 3.3 直接编辑工具

- **分割**：用一个或多个曲面把实体切开；
- **组合**：布尔加、减、交运算；
- **移动/复制实体**：平移或旋转实体；
- **删除/保留实体**：在多实体零件中清理不需要的实体；
- **移动面 / 删除面**：快速修改 imported 几何。

### 3.4 装配体基础

1. 新建装配体文件（`.sldasm`）。
2. 点击**插入零部件**，将已保存的零件加入装配体。
3. 使用**配合**关系约束零件相对位置。

常用配合：

| 配合 | 含义 | 示例 |
| --- | --- | --- |
| 重合 | 两个面/线/点重合 | 法兰盘贴合 |
| 平行 | 两个面平行 | 导轨安装面 |
| 垂直 | 两个面垂直 | 角码安装 |
| 同心 | 两个圆/圆柱同轴 | 轴与轴承 |
| 距离 | 两个面保持固定距离 | 垫片厚度 |
| 角度 | 两个面成固定角度 | 倾斜支座 |

装配体高级技巧：

- **随配合复制**：复制零件时一并复制其配合关系；
- **替换零部件**：用新零件替换装配体中的旧零件；
- **干涉检查**：检查零件之间是否存在干涉；
- **质量属性**：查看总质量、重心、转动惯量。

## 4. 导出文件格式

机器人队常用导出格式：

| 格式 | 用途 | 操作路径 |
| --- | --- | --- |
| `.stl` | 3D 打印、仿真 | 文件 → 另存为 → 保存类型选 STL |
| `.dwg` | 2D 工程图、激光切割 | 工程图 → 另存为 DWG |
| `.step` / `.iges` | 跨软件交换 | 文件 → 另存为 → STEP/IGES |

导出 STL 时注意：

- **偏差/角度**设置影响曲面精度，建议先使用默认；
- 如果是装配体，选择**保存所有零部件为单一文件**或**分别保存**，视 3D 打印需求而定。

## 5. AutoCAD 简介

队内 AutoCAD 使用相对较少，主要用于：

- 查看和简单编辑供应商提供的 2D 图纸；
- 激光切割路径检查与排版；
- 快速标注与出图。

新人以自学为主，建议掌握：

- 直线、圆、偏移、修剪、标注；
- 图层管理；
- 打印/导出 PDF。

## 6. 实战任务：带孔支架

### 任务目标

完成下图所示的简易支架，要求：

- 底板 80 mm × 60 mm × 5 mm；
- 立板 60 mm × 50 mm × 5 mm，与底板垂直；
- 立板中心有一个直径 10 mm 的通孔；
- 底板四角各有 1 个直径 4 mm 的通孔；
- 添加 2 mm 圆角（除安装孔边缘外）。

### 建议步骤

1. 新建零件，在前视基准面绘制底板草图（80×60 矩形），拉伸 5 mm。
2. 在底板顶面绘制 4 个 φ4 圆，使用**完全贯穿**拉伸切除。
3. 在底板侧面绘制立板轮廓（60×50），拉伸 5 mm。
4. 在立板正面绘制 φ10 圆，完全贯穿切除。
5. 选中需要圆角的边，添加 2 mm 圆角。
6. 使用**质量属性**查看零件质量。
7. 另存为 `.stl` 与 `.dwg`。

## 7. 今日小结

- SolidWorks 环境配置决定了后续建模效率，建议第一次使用时统一设置好单位和快捷键；
- 草图完全定义是参数化建模的基础；
- 特征命令按"拉伸/旋转 → 切除 → 阵列/镜像 → 圆角倒角"的顺序使用，建模思路最清晰；
- 装配体配合从"面-面"关系入手，逐步约束自由度。

## 8. 常见问题

### Q1：草图线条是蓝色的，无法拉伸？

蓝色表示草图欠定义。使用**智能尺寸**或**几何关系**将其完全定义后再试。

### Q2：装配体中零件配合后仍能动？

检查是否还有未约束的自由度。一个零件在空间中通常需要 3 个非冗余配合才能完全定位。

### Q3：导出 STL 后模型有破面？

检查是否存在零厚度几何、自相交面或未缝合的边。可先运行**检查**工具（工具 → 检查）。

### Q4：SolidWorks 启动很慢？

关闭启动时加载的插件：工具 → 插件，取消不常用插件（如 CircuitWorks、PDM 等）。
