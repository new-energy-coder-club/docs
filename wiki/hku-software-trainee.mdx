## 1. 编程工具

- **C / C++**：队伍主要使用的编程语言。
- **Linux**：机器人运行的操作系统，建议先掌握常用命令。
- **Git**：日常版本控制工具，程序员协作必备。
- **ROS（Robot Operating System）**：用于构建机器人应用的软件库与工具集。

## 2. 训练教程

原站点提供了一系列关于 Linux、Git 和 ROS 的培训教程，帮助新人掌握 Robocon 队伍中所需的编程技能。

### Linux 基础

- [SSH 远程连接](https://docs.m2stud.io/cs/linux/ssh/) — 学习 `ssh`、`tmux`、`sftp` 等远程开发与文件传输工具。

### Arduino / 嵌入式

- [Lab 1.3 — 增量式编码器与库](https://docs.m2stud.io/cs/arduino/abi/)
- [Lab 1.4 — 无刷电机与 CAN 通信](https://docs.m2stud.io/cs/arduino/can/)
- [补充：I2C 通信与传感器](https://docs.m2stud.io/cs/arduino/i2c/)
- [Lab 7.1 — Arduino digitalWrite / analogWrite](https://docs.m2stud.io/cs/arduino/07-L1-Arduino-digitalWrite-analogWrite/)

### ROS 2 实践

- [Assignment 1 — 使用 m2_joy 的手柄控制器](https://docs.m2stud.io/cs/ros/turtle_ps4/) — 在 turtlesim 中用 PS4 手柄控制小乌龟。
- [Assignment 2 — 自动路径跟随](https://docs.m2stud.io/cs/ros/turtle_path/) — 控制模拟机器人沿预定义路径移动。
- [Lab 6.3 — 使用 RViz 进行可视化](https://docs.m2stud.io/cs/ros_additional/06-L3-rviz/)

---

## 3. 学习建议

1. 先熟悉 **Linux 常用命令** 和 **Git 工作流**。
2. 完成 ROS 2 基础实验（topics、services、package、msg/srv）。
3. 逐步完成 turtlesim 系列作业，理解节点、话题、服务、launch 文件的使用。
4. 结合 NEC 实际项目代码，尝试在机器人上复现类似功能。
