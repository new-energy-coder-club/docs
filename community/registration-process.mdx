欢迎加入 **NEC 新能源开发者社区**！本页面将带你完成从报名到正式入社的全部步骤，预计耗时 **15-30 分钟**。

---

## 报名资格

| 条件 | 说明 |
|------|------|
| 身份 | 年满 18 周岁的高校在校学生、教师或工程从业者 |
| 技术方向 | 不限，机械/硬件/软件/视觉/运营/项目管理均可参与 |
| 时间投入 | 至少能保证每周 3-5 小时参与社区活动或项目 |
| 态度 | 愿意主动学习、积极提问、乐于分享 |

> 无需相关竞赛经验，社区为新人准备了完整的培训路径与导师指引。

## 报名流程总览

<Steps>
  <Step title="加入 QQ 交流群">
    加入 NEC QQ 交流群（群号：**479053780**），申请时备注：`姓名 + 学校/公司 + 技术方向`。
  </Step>
  <Step title="填写飞书信息收集表">
    扫码填写社区飞书收集表，完成成员信息登记与个人展板资料填写。
  </Step>
  <Step title="注册 Gitee 账号">
    前往 [Gitee](https://gitee.com/) 注册账号，这是社区代码托管与协作的主力平台。
  </Step>
  <Step title="Fork 主仓库">
    Fork [NEC 主仓库](https://gitee.com/darrenpig/new_energy_coder_club) 到自己的 Gitee 账号下。
  </Step>
  <Step title="完成首个 Good First Issue">
    从 `good first issue` 标签中挑选一个任务，提交你的第一个 Pull Request。
  </Step>
  <Step title="加入飞书组织">
    完成首个任务后，联系运营同学邀请你加入 NEC 飞书组织与内部群组。
  </Step>
</Steps>

## 第五步：完成首个 Good First Issue

这是新人的**考核任务**，也是你融入社区的最佳方式。

<CardGroup cols={2}>
  <Card
    title="浏览 Good First Issues"
    icon="search"
    href="https://gitee.com/darrenpig/new_energy_coder_club/issues"
  >
    在 Issues 列表中筛选 `good first issue` 标签，挑选一个适合你的任务。
  </Card>
  <Card
    title="新人任务指南"
    icon="map"
    href="/start-here/first-good-issue"
  >
    详细了解新人任务体系、标签规范与交付标准。
  </Card>
</CardGroup>

### 提交流程

<Steps>
  <Step title="认领任务">
    在选中的 Issue 下评论：`我想认领这个任务，预计在 3 天内提交 PR。`
  </Step>
  <Step title="创建分支">
    ```bash
    git checkout -b gfi/<issue-id>-short-title
    ```
  </Step>
  <Step title="完成并提交">
    按 Issue 要求完成内容，commit message 格式：`docs: 简短说明`
  </Step>
  <Step title="发起 Pull Request">
    向主仓库 `main` 分支发起 PR，描述中关联 Issue 编号。
  </Step>
</Steps>

<Note>
  首次 PR 建议选择 `type:docs` 标签的文档类任务，难度较低、上手更快。
</Note>

## 常见问题

<AccordionGroup>
  <Accordion title="报名有时间限制吗？">
    社区报名全年开放，但各赛季有集中招新期（通常在每年 5-6 月与 11-12 月）。集中招新期会有更多培训资源与导师支持。
  </Accordion>
  <Accordion title="我不确定自己适合哪个方向，怎么办？">
    可以先浏览 <a href="/start-here/newcomer-guide">新人入门指南</a> 的技能基线，了解各组别特点。也可以在 QQ 群中咨询各方向学长，他们会根据你的背景给出建议。
  </Accordion>
  <Accordion title="报名需要缴费吗？">
    不需要。NEC 社区是开源、免费的社区组织，不收取任何形式的报名费或会费。
  </Accordion>
  <Accordion title="我已经有竞赛经验，可以跳过 Good First Issue 吗？">
    首次 Good First Issue 不仅是技术考核，更是让你熟悉社区协作流程（分支规范、PR 模板、Review 流程）的必经环节。有经验的成员可以选择 `difficulty:3` 以上的任务。
  </Accordion>
  <Accordion title="飞书收集表提交后没收到反馈？">
    收集表由运营同学定期处理，通常在 2-3 个工作日内完成。如超过一周未收到反馈，请直接在 QQ 群中 @运营 查询。
  </Accordion>
</AccordionGroup>
