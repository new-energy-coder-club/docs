## 提问指南

### 好的提问

✅ **提供上下文**
```
我正在开发底盘控制程序，使用 STM32F407 + FreeRTOS。
遇到了以下问题：电机 PWM 输出不稳定...
```

✅ **提供详细信息**
```
环境：STM32CubeIDE 1.12.0，HAL 库
错误信息：HardFault_Handler 被触发
已尝试：检查堆栈溢出、增加任务栈大小
```

✅ **提供复现步骤**
```
1. 启动电机控制任务
2. 发送速度指令 1000 RPM
3. 运行约 10 秒后发生错误
```

### 避免的提问

❌ **过于笼统**
```
我的代码不工作，怎么办？
```

❌ **不提供上下文**
```
这个错误什么意思？
[截图只有错误代码，没有代码]
```

❌ **不反馈结果**
```
[提出问题后，别人给出建议，但提问者不再回复]
```

## 行为规范

### 应该做的

- 友善对待每一位成员
- 分享你的知识和经验
- 对他人的帮助表示感谢
- 使用清晰的标题描述问题
- 在问题解决后分享解决方案

### 不应该做的

- 使用不友善或攻击性的语言
- 发送与主题无关的内容
- 重复发送相同问题
- 私下索要代码而不自己尝试
- 泄露他人的隐私信息

## 成为版主

如果你对社区有热情，愿意帮助他人，可以申请成为版主：

1. 在社区活跃 3 个月以上
2. 有帮助他人的记录
3. 提交版主申请
4. 通过核心团队审核

版主职责：
- 维护讨论秩序
- 回答问题
- 整理优质内容
- 组织线上活动
